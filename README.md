# 🍽️ Zomato Data Analysis Project

## 📌 Overview
An end-to-end data analysis project on the Zomato dataset using **Excel, MySQL, Power BI, and Tableau** to uncover business insights and build interactive dashboards.

## 🛠️ Tools Used
- Microsoft Excel
- MySQL
- Power BI
- Tableau

## 📂 Project Files
- 📊 Excel Analysis
- 🗄️ SQL Analysis
- 📈 Power BI Dashboard
- 📉 Tableau Dashboard
